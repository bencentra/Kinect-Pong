kinect-pong
===========

An implementation of Pong in a C# WPF application, using the Microsoft Kinect for controls.    

Uses [WpfKinectHelper] (https://github.com/bencentra/WpfKinectHelper) for interfacing with the Kinect. 

Made by Ben Centra
